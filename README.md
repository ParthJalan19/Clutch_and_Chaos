Overview

Clutch & Chaos is a premium automotive web application designed to showcase high-performance sports cars and motorcycles through a modern and engaging interface. The project emphasizes responsive design, smooth interactions, and clean UI principles to deliver an intuitive browsing experience across all devices.

This project was built to strengthen front-end development skills and gain hands-on experience with modern frameworks and utility-first styling.

Project Goals

Build a visually appealing automotive website

Implement responsive layouts for multiple screen sizes

Practice component-based development using React

Improve UI consistency using Tailwind CSS

Enhance user experience with category-based filtering

Key Features

Separate sections for sports cars and motorcycles

Category-based filtering to easily explore vehicles

Fully responsive design for desktop, tablet, and mobile

Clean, modern layout with consistent styling

Fast load time and smooth navigation
